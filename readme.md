Study Smart

A modern web application built using React, TypeScript, and Vite with a clean and scalable folder structure. This project is designed to provide a fast, responsive, and maintainable frontend application.

📌 Project Overview

Study Smart is a frontend web application that follows best practices in modern web development. It uses Vite for fast builds, TypeScript for type safety, and Tailwind CSS for styling.

🛠️ Technologies Used
React
TypeScript
Vite
Tailwind CSS
Playwright (for testing)

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/your-username/study-smart.git
Navigate to the project folder:
cd study-smart
Install dependencies:
npm install
Run the development server:
npm run dev
🌐 Running the App

After running the server, open:

http://localhost:5173/
